# Appium Test Cases

## 1. Auth Test
- Pre-condition: App installed, emulator running
- Steps:
  1. Open app
  2. Enter email & password
  3. Click login
- Expected Result:
  - Navigate to Home screen

## 2. Cart Test
- Pre-condition: User logged in
- Steps:
  1. Open product
  2. Add to cart
  3. Open cart
- Expected Result:
  - Product appears in cart

## Tools
- Appium
- Android Emulator
- Node.js
